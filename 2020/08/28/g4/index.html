狮鹫的力量

发表于 2020-08-28 分类于 github 阅读次数：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

  最近吧复古了，看了不少魔兽视频，xiaoy解说还是挺有意思的，翻了翻电脑你还别说我还真翻出来了以前的war3录像，当年兄弟们都没结婚，单身男青年最喜欢的就是抄机了，随着sahu都快成家了，我们的war3兄弟团已经解散很久了。

  回到replays，这盘是2009年4月30日22点在vs平台上打的录像，一晃10年就这么过去了，沧海桑田，物是人非，好像已经2-3年连聚会碰头都没有碰了。战斗是在Gnollwood上打响的，这张狗头人的地图也是我们最熟悉的，很平衡，什么东西都有，酒馆、商店、地精实验室、生命/魔法泉都有，唯一的缺点就是分矿太少了，3v3基本不够用。

3v3开打，sahu、贵贵和我一对，对三个路人，随机出来sahu随到了他最擅长的人类，贵贵随机到了他最不擅长的亡灵，我也是亡灵。对面和我们一样也是人鬼鬼，还是挺平衡的。

sahu首发大法师开始练级，标准的步兵farm，贵贵出恐惧魔王，吸血狂狗的路数，他们都是近战我只能出远程了，我出了DK加蜘蛛的策略。

对面人族首发mk，亡灵都是dk出场没有太大的变化。贵贵的恐惧魔王带着5条小狗就直奔主题往对面亡灵家里去了，然而他的攻击对象是金矿，我感觉不靠谱，如果能选择拆商店或者杀农民大概会好一些吧。

显然这几条狗在对方有冰塔的情况下金矿是拆不掉的，赔掉一本tp狗死光，没得到任何好处，血亏。另一边sahu也开始进攻另一家亡灵。

对方基地建设紧密，排放整齐，没有机会，这时接到了贵贵的求救，刚才血亏之后对方dk显然不在乎什么穷寇莫追，就想要痛打落水狗，步步紧逼，对方人族mk带着步兵也是一起冲进了贵贵的基地，贵贵正在升级2本基地，家里基本啥也没有。

sahu的人族还是靠谱的轻松打退对手，我带着蜘蛛和sahu剩下的部队一起重新杀往右侧的亡灵，不能让人闷声发展，该骚扰的必须骚扰。

这时sahu家里暗搓搓的放下双狮鹫笼，开始暴狮鹫，于是我们连续两波进攻都被女巫的减速搞的很惨，蜘蛛损失惨重，几乎是点一个死一个，没有驱散在这种情况下太劣势了。

那边贵贵终于雄起了一把，睡接围住了对方的dk，可惜人家有tp，于是再次被对方搞了个全军覆没。

太嚣张了，把贵贵搞的这么惨，怎么也要复仇吧，于是大军上路了！

看到大量sahu的狮鹫，铺天盖地，在狮鹫的攻击下，英雄也是脆的一塌糊涂，分分钟倒，错，是秒倒。

不过在大战的时候，对方也留了一手，派了2个冰龙来换家，把sahu家拆的七七八八，双方等于在比赛谁拆的更快更有效率了。

这时发生搞笑一幕，那石头人也耐不住寂寞到我家来玩了一把..

在各自拆掉了2家之后，终于要大决战了！sahu的狮鹫一马当先，对方的冰龙直接骑脸，瞬间秒掉了sahu的大法师。

在对面无数防御塔的帮助下，顶住了一波进攻，sahu又暗搓搓在角落里造了3-4个民房，补足了人口，借我的钱买活了大法师，贵贵也带着冰龙一起冲冲冲，再次杀入这把对方只能gg了。

这盘Replay展示了sahu人类的功底还是非常强悍的，战术多变，表现抢眼，也展示了贵贵后期果断出冰龙的优秀表现，我等这种菜鸡多出点顶级兵可以大幅度的增加人口，效率并不差。
