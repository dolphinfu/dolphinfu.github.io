寝室对抗赛

发表于 2020-09-04 分类于 github 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

  人这一辈子除了和亲人外，大约只有和宿舍的老铁们住在一起的时间最久了吧，大学同房四年，谁打呼噜谁脚臭都一清二楚，这种情况下的默契程度毫无疑问也体现在游戏中，来来看看寝室对抗赛的replays吧。

这场战斗是来自2009年2月6日半夜10点53分，216寝室的三位大将——zz、sl、贵贵对抗我们其它宿舍的杂牌军——sahu、小问号和我，理论上讲双方水平差不多，我方按理还强一些，不过最后的结果也是大跌眼镜，几乎一场完败，团战就没赢过。

开局sahu的兽人，小问号的亡灵和我的人类，马马虎虎的组合，小问号的亡灵用的不错，sahu的兽人基本是他垫底的种族，我也不太喜欢人类。对面216战队贵贵的亡灵还行，zz的亡灵凶猛，sl最爱的兽人，像我们这样的水平的对抗，亡灵还是占有一定优势的，冰龙还是挺强悍的。

贵贵的打法上一篇已说过恐惧魔王的睡狗，后期不出疯狗出冰龙，不过这盘没出冰龙出的是毁灭，基本也差不多，zz是蜘蛛流，出兵速度快，操作好，后期出了冰龙。sl的牛头人带g后期出猎头海，专注于远程攻击。

我们打法比较混乱，我是男女巫破法加火枪，sahu的狼骑，小问号的蜘蛛加憎恶。

双方正式开打，也没有相互骚扰而是很友好的分别进行练级，增强英雄的等级。

分别干掉了一个分矿点，狗头人确实也不强，一家带几个小兵就可以搞定，这地图野兵强弱太分明，分矿说实话比蓝胖子的食人魔要容易一些。

继续练级我们干掉了另一块分矿，他们集合做掉了中间的红龙，蜘蛛貌似都没升捕捉，直接用猎头上了，也是挺暴力的。

好了地图上的点都练得七七八八了，总要开打了吧，216队从下面摸进了我的主基地，连门口的房子都拆了，也还好门口放了个房子增加了些许视野，赢得了一些时间，我们绕了一圈来防守，我的部队走大了，尤其是大法师瞬间陷入包围，一着不慎满盘皆输，火枪也损失不少。

总算打退了对方，但是损失惨重，英雄阵亡几个，小问号还被sl拆了分矿，局势已经极其不利了。

在这里我们又团战了一把，这次更糟糕，兵力不够阵型也不好，sahu的狼骑太脆皮了，都打不过吸血狗，保护卷轴也用早了，部队还没有集中就用了，后排的基本没吃到，团补也用的不好，这仗打亏了很多。

好了终于被拆家了，一波混战后全军覆没，差不多是时候gg了。

我们仨统统垫底，配合也没打好，兵种组合和英雄组合都不太给力，也没有对抗毁灭和冰龙的部队，总体来说是一场完败。对方一个宿舍的知根知底，该近战的近战该远程的远程，配合默契，行动一致，zz也用上了最擅长的亡灵，充分发挥了水平，打出了他们寝室的气势和风格，给点掌声。
