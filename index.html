海口东坡湖

发表于 2023-03-02 分类于 github
本文字数： 707 阅读时长 ≈ 1 分钟

  在海口，一个早春的傍晚，我在海南大学东坡湖边，欣赏水鸟扑腾，坐看夕阳西下。
  海南大学海甸校区坐落在海南省海口市美兰区海甸岛。海甸岛是海口市的一个城中岛，与市区仅一水之隔，不仔细观察甚至看不出是一座岛屿。海南大学则是一所历史悠久的大学，海甸校区是海南大学主校区，校园环境优美，设施先进，建筑风格独特、学术氛围浓厚，可以说是一处天然的公园。
  校区最大的亮点就是东坡湖。北宋大文豪苏轼因“乌台诗案”流放海南三年，东坡先生一生豁达壮丽，为海南教育留下了诸多贡献，海南历史上第一位举人姜唐佐就是东坡门生。海南大学以东坡之名称此湖确实恰如其分。
  东坡湖的中央有一座岛屿，绿荫覆盖，鸟儿以此为居所，是它们的天堂，时而可见白鹭在湖水上飞翔、捕食，它们的身影在夕阳的照耀下格外优美。东坡词《南乡子·晚景落琼杯》云：“一阵东风来卷地，吹回，落照江天一半开。” 落日的余晖从云缝中洒落出来，把半边天染红，连湖水也红了一半，景色如是。

  坐在东坡湖边，四周有着优美的绿色植被，如红树林和海芒果等。即使在二月，这些树木和花草仍然保持着生机和活力，让人不禁感叹大自然的魅力。苏轼《和陶游斜川》诗云：“春江绿未波，人卧船自流。我本无所适，泛泛随鸣鸥。”我虽没有坐船荡漾在东坡湖上，却也感受到了东坡先生的自然之乐、闲适之感。

  湖的远方，太阳落下的地方正是海口标志性的景物——世纪大桥。世纪大桥主桥长636米，两座主塔高107米，高耸入云，是海南人民勤劳与智慧的结晶，在这一刻现代科技文明与古典景色和谐的结合在了一起。

海口雷琼地质公园

发表于 2023-03-01 分类于 github
本文字数： 1.2k 阅读时长 ≈ 1 分钟

  之所以选择来到海口，一方面是因为海南的美食和热带风光令我着迷，数年前在三亚，迷人的热带雨林，慵懒的沙滩都给我留下了美好的记忆。另一方面是我一直想去探索一下海口的雷琼火山地质公园。在这次旅行中，我感受到了一种与热带雨林、沙滩海水完全不同的海南风情。

  我选择在海口市区内乘坐游1路前往地质公园，上车需刷卡15次也是让我十分震惊，几乎每一位上车的游客都是如此，算是一段意外的小插曲了。

  愉快的来到了位于海口市南部的雷琼世界地质公园海口园区，这里是4A级景区，离海口市区仅15km，雷琼火山群是非常大的一个概念，包括广东南部和海南均有分布，此地是位于琼北的海口园区。说起地质公园的自然景观自是十分独特，它曾经是一个火山群，拥有多座火山口，是距今2.7万年至100万年间火山喷发所形成的死火山群，可以说是一处天然的地质公园。最大的一座马鞍岭火山口如今已经形成了一座山丘，是世界上保存完整的死火山口之一，也是海口地区的制高点。我沿着山丘上的小路一路向上前行，茂密的林木中，散布着许多稀奇的火山石和火山喷发后留下的遗迹，如今这些曾经恐怖的熔浆早已成为了黑色的大地，而黑色的大地又被绿树所覆盖。仔细观察那些有着上万年历史的火山岩石，清晰可见它“生命”中曾经存在的流动的痕迹。

  接下来，我登上了海口的“制高点”——马鞍岭火山口。从这里可以远眺整个海口市区，景色优美。此处位于海口市区的东南方向，海拔约 222 米，是这片土地上的一个重要地标。山下热带树林郁郁葱葱，空气清新。登顶之后就要下山了，这里的下山可不是原路返回，而是进入最刺激的火山口深处，马鞍岭火山口形似马鞍形，由两座火山锥——风炉岭、包子岭组合形成马鞍的形状，这里参观的是风炉岭火山，火山口直径220米，深90米，像一个巨大的峡谷，茂密的树林挡住了视线，一眼望不到底。经过陡峭的火山石山路终于到达了火山喷发的地方，所谓沧海桑田，跺跺脚丝毫没有晃动的感觉，此地已是实地。映入眼帘的榕树从山顶一路向下生长，宛如倒挂的森林。

  最后，我深刻地认识了火山对于海南岛上的重要性。火山岩不仅是海南岛形成的重要地质标志，而且被广泛用于雕刻艺术品等方面。海南人将火山石雕刻成了许多精美的火山石艺术品，也是别具特色。

  苏轼曾贬谪海南，作诗描写过同样由火山石玄武岩构成的儋州山丘，其《儋耳山》诗云：“突兀隘空虚，他山总不如。君看道傍石，尽是补天馀。”苏轼与众不同的才华如有补天之能，却遭遗弃，只能在海南谪居，如同多余了一般。诗中借景喻心中的愤慨和烦恼，每每读到此处不由得泪眼朦胧，时而又会想到何等奇妙的石头，能有补天之状？能让苏轼自比之物，想来绝非凡品。如今亲临海南，虽未到儋州儋耳山，但看到这山中火山奇石，与众不同，如天赐一般，其独特之处方能匹配苏轼的补天之才！

转化表格为文本

  转化表格为文本

今天也是有点意思，遇到个问题，就是怎么处理一个从其他软件转化出来的word文本，而这个word文件吧，居然全是表格形式的，表格之间还有空行。
我试了试全选表格，那个转化为文本是灰色的，不让用，这倒是个麻烦事。
想了想还是用宏吧，vba我没用过，研究了半天，写了一段代码好像还能用，在不知道会不会有bug了。

Sub ConvertAllTables()

Dim tempTable As Table

    Application.ScreenUpdating = False

    ActiveDocument.DeleteAllEditableRanges wdEditorEveryone
    For Each tempTable In ActiveDocument.Tables
        tempTable.Range.Editors.Add wdEditorEveryone
        tempTable.ConvertToText Separator:=wdSeparateByParagraphs, _
        NestedTables:=True
    Next
    ActiveDocument.SelectAllEditableRanges wdEditorEveryone
    ActiveDocument.DeleteAllEditableRanges wdEditorEveryone
    Application.ScreenUpdating = True
End Sub
