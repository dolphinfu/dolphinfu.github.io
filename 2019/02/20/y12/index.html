人口失衡

发表于 2019-02-20 分类于 yiji 阅读次数：
本文字数： 682 阅读时长 ≈ 1 分钟

  这几天听说魔兽事件人口已经完全失衡了，汗，基本上一直也都是不过人多，我觉得中国玩家联盟大幅度减少应该是从tbc和wlk开始的，但是这时候还有救，起码联盟还是有人的，那么什么时候联盟玩不下去了？当然就是熊猫人了，坑爹的日常你非做不可不做不行，想跑都跑不掉，这时候阵营优势无限放大，弱势方惨不忍睹，一点补偿都没有谁还玩啊，大家又不傻。然后大跨服是压死骆驼最后一根草。

  我玩了十几年了，60级，联盟比部落多，因为精灵好看，亡灵虽然天赋好毕竟有的人有密集恐惧症…联盟还是圣骑这种信仰职业，并且pve直到taq之前都有极为可观的优势，所以联盟人更多。

  tbc血精灵强势加入部落，部落人口增加，双方均势。

  wlk死亡骑士和巫妖王的魅力，部落人口反超，不过好在还能玩的下去，世界战场人少的有buff弱势群体勉强还能玩的下去。

  85级可能是我最不认识副本大门的一个版本了，后期加入了坑爹的日常，好在场景转换还算好，反正不做这个日常的敌人你也碰不到，但是联盟进一步衰落是不争的事实。

  熊猫人开始就完蛋了，日常又难做又打架吃饱了撑的吧？还出了个杀人拿牌子…最后的活路都不给。

  后面两个版本我们服联盟已经只有部落的十分之一了，也恢复不起来了。不过有的联盟部落差不多的服因为搞了大跨服，本来还能活，结果也过不下去了…

  这个版本虽然给了pve选项，但是已经错过了时机，人已经少到pve活动开展都困难的程度还玩啥？
