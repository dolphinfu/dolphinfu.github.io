粤港澳大区

发表于 2019-02-27 分类于 yiji 阅读次数：
本文字数： 344 阅读时长 ≈ 1 分钟

  最近粤港澳大区沸沸扬扬，说实在的行政不隶属办事很困难，广州内部搞搞的一直都是做的非常不错的，城市之间距离越拉越近，但是加上香港？我不看好，就比如广州修地铁去深圳当然比上海修地铁去苏州要快得多，但是广州修大桥去香港…绝对比上海修大桥去宁波难得多得多。这个世界就这么现实。

  上海城市圈只要服侍好杭州，砸足够的钱补助给南京，虽然事情麻烦终究还是能进行的，你看周围一圈哪个不想和上海更紧密，而上海会不欢迎哪个？这是自由恋爱啊亲。困难当然也有，杭州始终不能很好的融入长三角，杭州语言和长三角其他城市有些不太一样，文化上也有些隔阂，近年来杭州发展确实很快速但和周边的结合度还是要差一点的，上海产业链也很少首选杭州的，两个核心城市之间互动有点差，除了旅游…
