海口东坡湖

发表于 2023-03-02 分类于 github
本文字数： 707 阅读时长 ≈ 1 分钟

  在海口，一个早春的傍晚，我在海南大学东坡湖边，欣赏水鸟扑腾，坐看夕阳西下。
  海南大学海甸校区坐落在海南省海口市美兰区海甸岛。海甸岛是海口市的一个城中岛，与市区仅一水之隔，不仔细观察甚至看不出是一座岛屿。海南大学则是一所历史悠久的大学，海甸校区是海南大学主校区，校园环境优美，设施先进，建筑风格独特、学术氛围浓厚，可以说是一处天然的公园。

  之所以选择来到海口，一方面是因为海南的美食和热带风光令我着迷，数年前在三亚，迷人的热带雨林，慵懒的沙滩都给我留下了美好的记忆。另一方面是我一直想去探索一下海口的雷琼火山地质公园。在这次旅行中，我感受到了一种与热带雨林、沙滩海水完全不同的海南风情。

海口骑楼老街

发表于 2023-02-25 分类于 github
本文字数： 1.1k 阅读时长 ≈ 1 分钟

  骑楼老街是海口市的一条历史悠久的古老街道，也是海口市的著名旅游景点之一，初建于1849年的海口骑楼老街是现今国内骑楼建筑保留规模较大、保存基本完好、极富中西特色的历史文化街区。集中分布在海口市龙华区和美兰区的中山路、得胜沙路、新华路等。。骑楼老街是类似于步行街的老城区，东边都是步行街，西边则可以开车包括电瓶车，所以西边对行人不太友好，我主要看的也是东侧的景区。
