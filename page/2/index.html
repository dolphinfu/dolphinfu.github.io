转化表格为文本

  转化表格为文本

今天也是有点意思，遇到个问题，就是怎么处理一个从其他软件转化出来的word文本，而这个word文件吧，居然全是表格形式的，表格之间还有空行。
我试了试全选表格，那个转化为文本是灰色的，不让用，这倒是个麻烦事。
想了想还是用宏吧，vba我没用过，研究了半天，写了一段代码好像还能用，在不知道会不会有bug了。

Sub ConvertAllTables()

Dim tempTable As Table

    Application.ScreenUpdating = False

    ActiveDocument.DeleteAllEditableRanges wdEditorEveryone
    For Each tempTable In ActiveDocument.Tables
        tempTable.Range.Editors.Add wdEditorEveryone
        tempTable.ConvertToText Separator:=wdSeparateByParagraphs, _
        NestedTables:=True
    Next
    ActiveDocument.SelectAllEditableRanges wdEditorEveryone
    ActiveDocument.DeleteAllEditableRanges wdEditorEveryone
    Application.ScreenUpdating = True
End Sub
