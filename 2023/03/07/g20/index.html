海口五公祠

发表于 2023-03-07 分类于 github 阅读次数：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

  五公祠，闻名可知此处乃是纪念五位名人所建之祠堂，但初来乍到并不知五公是何人，就当是逐步探索其人其事了。此处祠堂造型十分奇特，整体建筑是向下走的，门口一个大牌坊，也不知道是不是古迹，看着不像是，而建筑都在下面，门口反而是整个五公祠的最高处，往下倾斜的。让人惊呆的是门口就一个小摊贩卖票，着实不像是一个旅游景点，要不是门票看着很正规，也有检票口，我差点当是被黄牛骗了。
  里面不算太大，中间是个小湖，四周一圈以及湖中有个小岛，湖的西侧则是楼房，也就是五公祠的主体建筑，也是这里最有古迹的地方，顺便也就揭开了五公的来龙去脉。

  海口五公祠是海口历史最悠久的古典园林建筑群，始建于北宋年间。和江南园林不同，五公祠代表的是海南的贬谪文化，在这里供奉的是唐代李德裕，北宋末年李纲，南宋李光、赵鼎、胡铨五人，这五人皆有贬谪海南的历史，在当地为社会的发展贡献过力量，因此被海南人民所敬仰和感激。这五公所在的楼又称“海南第一楼”，是一座红色的木制的门楼，两层高，别具特色。门前的对联写着：五公英烈气，千古海南潮。庭院内还有五公的石像和他们的简介。
  李德裕我比较熟悉，是晚唐时期重要的宰相，也是牛李党争的主角之一，著名诗人李商隐对李德裕非常推崇，称之为“万古良相”，而李德裕在对会昌年间晚唐中兴起到了关键的作用，会昌年间大破回纥，迎回太和公主，也算是一段全盛时期了。遗憾的是最终李德裕的结局并不好，最终被贬崖州，逝世于此。
  李纲则是北宋末著名的将领，主持了东京保卫战，取得胜利，无奈最终被贬，失去李纲后北宋无力守住京城，最终北宋灭亡。李纲在南宋时期也曾被贬琼州，不过时间并不长，但其忧国忧民的精神还是受到海南居民的爱戴。
  李纲、李光、赵鼎、胡铨也并称为南宋四名臣，皆为南宋初年忠贞之士，他们以坚定的立场和卓越的才能，为南宋的稳定作出了不可磨灭的贡献。然而，他们的正直却与当时的权臣秦桧形成了尖锐的对立，因此相继遭到贬谪，甚至被迫害致死。尽管如此，他们的忠义精神和高尚品格却永远铭刻在历史的丰碑上。

  在海南第一楼边上，有苏公祠，是一座明代所建的高楼，为纪念苏东坡所建，还存有苏东坡所开凿的浮粟泉和洗心泉，其中浮粟泉被称为“海南第一泉”。苏公祠中书有“南溟奇甸”就是海南的代指。苏公祠今天有活动，有一群学生穿着古装，在学习苏轼到海南时，姜唐佐等海南子弟拜师东坡学士的场景，文化需要传承，东坡精神将世代在海南流传。苏东坡在海南时，得知当地群众饮水不易，因此开凿了两口泉水，俘粟泉和洗心泉，其中俘粟泉水质更好，逐渐成为了此地的名胜，今天还能看到苏东坡所亲手打凿的泉水。

  苏公祠边上还有两伏波祠，是为纪念西汉的路博德、东汉的马援两位伏波将军而建，两位伏波将军平顶南越叛乱，为海南的和平做出重要贡献，宋以后就已经是海口重要的祭祀庙宇，清代迁至五公祠，将这些对海南有很大贡献的名人们放在一起。所以五公祠并不仅有五公，还是海南文化的集成。
