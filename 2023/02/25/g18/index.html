海口骑楼老街

发表于 2023-02-25 分类于 github 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

  骑楼老街是海口市的一条历史悠久的古老街道，也是海口市的著名旅游景点之一，初建于1849年的海口骑楼老街是现今国内骑楼建筑保留规模较大、保存基本完好、极富中西特色的历史文化街区。集中分布在海口市龙华区和美兰区的中山路、得胜沙路、新华路等。。骑楼老街是类似于步行街的老城区，东边都是步行街，西边则可以开车包括电瓶车，所以西边对行人不太友好，我主要看的也是东侧的景区。

  这一片老城区的建筑最早可以追溯到19世纪中，大多数是20世纪30年代至50年代的，拥有独特的海南风格。所谓骑楼，就是一种近代东南亚各国以及我国两广、海南等南方地区所流行的商住两用建筑，就是一楼是店面，二楼以上就是住户，大多数三层，最高的看上去有五层。和一般的沿街商铺不同，骑楼的店门口比二楼要稍微缩进去一些，因此下雨天如果沿着店面走过去可以挡风遮雨，现在来看还可以停助动车..这种设计被称为柱廊相连，骑楼相依。当然这样的好处就是门面前可以挂两层牌匾，另外会多两根柱子，这些柱子饱经风霜，也可以看出早年的一些艺术设计。


  骑楼老街最大的看点就是这些骑楼建筑，因为这是早期中国人下南洋经商后回海口建设的，所以风格多样，有传统中国式设计，有哥特式，巴洛克式，或者东南亚风格的设计，总体是一种中欧混合式的风格，各种当年流行的设计都有所体现，故而使得这里也是一处小的万国建筑博览。看各种房屋的特色我觉得可以关注窗户和楼顶的女儿墙，中式的骑楼一般开有两扇方的窗户，窗户几乎没有装饰，这样的建筑其实比较少，算是一个基本款，都是比较老式的房子了。巴洛克式骑楼的窗户有上弧，弧度扁平，女儿墙比较复杂高大。哥特式建筑和巴洛克区别是窗户弧度更大，但女儿墙要简陋的多，有罗马柱的简单档墙。还有种南洋风格的，其实和巴洛克很接近，但女儿墙上开个洞，据说是因为南洋台风非常多，靠这个风孔来保证屋子的安全。


  除了建筑外，这里还有许多小店和餐馆，让游客可以品尝到当地的美食。在这里，我尝到了当地的传统小吃——椰子饭，味道么，和夜市吃的差不多吧，真的要吃本地特色还是要大众点评上好好找一找的了。这里还有一些当地的手工艺品。这些手工艺品都是由当地的手工艺人亲手制作的，以海南特色的产品为主，骑楼老街本身也是海口市最古老的商业街之一。

  这次来骑楼老街也非常幸运的是遇到了妈祖巡游海口骑楼老街活动，虽然去的时间晚了错过了妈祖金身，但却也看到了很多本土的活动，像舞狮等活动，十分热闹，气氛活跃，敲锣打鼓，喜气洋洋，也为新的一年平添了很多喜气。
